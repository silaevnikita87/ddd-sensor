האתגר האזרחי

רחפנים נכנסים למרחב האזרחי מהר יותר מהיכולת לזהות אותם.

מערכות גילוי קיימות יקרות, נקודתיות ודורשות חומרה ייעודית. לקהילות, מוסדות ותשתיות אזרחיות אין כלי נגיש להתרעה מוקדמת.

// עלות

חומרה יקרה

מכ״ם או חיישנים אקוסטיים ייעודיים עולים עשרות אלפי ש״ח לנקודה.

// כיסוי

נקודות בודדות

פתרון יחיד מכסה שטח מצומצם; פריסה רחבה אינה כלכלית.

// נגישות

לא בהישג יד

הגנה היקפית נשארת מחוץ להישג ידם של גופים אזרחיים.

ארכיטקטורה

ארבעה שלבים — מצליל למיקום להתרעה.

כל טלפון ברשת מקליט אודיו גולמי מסונכרן. הענן מצליב את ההקלטות, משלש את המקור, ומפיץ התרעה למפה חיה.

// 01

האזנה

טלפונים מקליטים אודיו גולמי (ללא עיבוד שמשמיד את אות הרחפן).

// 02

סנכרון

מחיאת כף משותפת מכיילת את הפרשי השעון בין הטלפונים למילישנייה.

// 03

שילוש

הפרשי זמן־הגעה (TDOA) בין שלושה חיישנים מצמצמים את המקור לנקודה.

// 04

התרעה

זיהוי במספר חיישנים בו־זמנית = אמון גבוה → התרעה ומיקום על מפה חיה.

עיקרון מנחה: לזהות ולהתריע — לא ליירט. כלי קהילתי להתרעה מוקדמת.
הפיזיקה

קול נע באוויר במהירות ידועה. זה כל הסוד.

כשרחפן משמיע צליל, הוא מגיע לכל חיישן בזמן מעט שונה. הפרש הזמן מקודד את ההפרש במרחק — וקווי ההפרשים נחתכים בדיוק במיקום המקור.

Δt × 343 m/s = Δdistance  →  position

  • ›שלושה חיישנים נותנים מיקום דו־ממדי; חיישן מוגבה מוסיף גובה.
  • ›הדיוק הגבוה ביותר כשהמקור בתוך מערך החיישנים או סמוך לו.

פעיל

הקלטה מסונכרנת

אפליקציה ילידית שמקליטה אודיו גולמי מסונכרן ומעלה אוטומטית לענן.

פעיל

מנוע שילוש TDOA

אומת בסימולציה ועל הקלטות אמיתיות — דיוק תת־מטרי בתוך המערך.

פעיל

מסך התראה חי

התראה רב־חיישנית בזמן אמת, רשימת חיישנים מחוברים, ופיד זיהויים.

פעיל

מפה גאוגרפית

חיישנים במיקום GPS אמיתי ואזור איום משוער — מתמקדת אוטומטית על הזיהוי.

שלב נוכחי: פיילוט ואימות. כיוונון ביצועי הגלאי מול נתוני רחפן אמיתיים הוא תחום הפיתוח הפעיל.

האפליקציה

הפכו את הטלפון שלכם לחיישן.

התקינו את אפליקציית CyberDuck, הצטרפו לסשן הקלטה מסונכרן, והפכו לחלק מהרשת. עיבוד מקומי במכשיר — האודיו לא נשמר ולא נשלח לשום מקום.

01 הורידו את קובץ ה-APK ופתחו אותו בטלפון אנדרואיד.
02 אשרו התקנה ממקור לא ידוע (הגדרה חד-פעמית).
03 פתחו את האפליקציה, תנו הרשאת מיקרופון — והצטרפו לסשן.
